# Retail Sales & Profit Analysis (SQL)

## Overview
This project analyzes mock retail financial data to uncover revenue trends, expense drivers, and profit margin categories using SQL.

## Skills Demonstrated
- SQL Joins between tables
- Aggregations with SUM and GROUP BY
- Use of DATEPART function to analyze by quarter
- Conditional logic with CASE WHEN for profit margin categories
- Window functions for rolling averages
- Basic data creation and inserts for testing

## Sample Output
- Total revenue, expenses, and profit by department and quarter
- Profit margin categorized as High, Medium, or Low for quick insights
- Rolling average profit trends over time by department
